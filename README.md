## Автор
**Агарков Дмитрий**

## Описание проекта
Training Planner — это графическое приложение для планирования и учёта тренировок. Позволяет добавлять, редактировать, удалять тренировки, фильтровать их по типу и дате, а также сохранять/загружать данные в формате JSON.

## Функциональные возможности

### Основные функции
- ✅ **Добавление тренировок** с указанием даты, типа и длительности
- ✅ **Просмотр списка** всех тренировок в удобной таблице
- ✅ **Редактирование** существующих тренировок
- ✅ **Удаление** отдельных тренировок или всех данных
- ✅ **Фильтрация** по типу тренировки
- ✅ **Фильтрация** по дате
- ✅ **Автоматическое сохранение** данных в JSON файл
- ✅ **Экспорт/импорт** данных в JSON
- ✅ **Статистика** (общее количество тренировок и длительность)

### Проверка корректности ввода
- Дата проверяется на соответствие формату `ГГГГ-ММ-ДД`
- Длительность должна быть положительным числом
- При некорректном вводе выводятся понятные сообщения об ошибке
